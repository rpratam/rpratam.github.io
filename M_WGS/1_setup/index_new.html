Panduan Instalasi WSL2 pada Windows

Panduan ini menjelaskan langkah-langkah untuk menginstal Windows Subsystem for Linux versi 2 (WSL2) pada sistem operasi Windows. Dengan WSL2, pengguna dapat menjalankan lingkungan Linux secara native di Windows dengan kinerja yang lebih baik dan kompatibilitas kernel Linux yang lebih tinggi.

Prasyarat

Sebelum melakukan instalasi WSL2, pastikan:

  • Versi Windows: Sistem operasi Windows adalah Windows 10 (versi 2004 dan lebih baru) atau Windows 11.
  • Virtualisasi: Hardware mendukung virtualisasi dan telah diaktifkan di BIOS.
  • Koneksi Internet: Diperlukan untuk mengunduh komponen dan distribusi Linux dari Microsoft Store.

Langkah-langkah Instalasi

1. Periksa Versi Windows

Pastikan Windows yang digunakan memenuhi syarat. Buka Command Prompt atau PowerShell dan jalankan:

winver

2. Aktifkan Fitur WSL dan Virtual Machine Platform

Buka PowerShell dengan hak istimewa Administrator, kemudian jalankan perintah berikut:

Aktifkan WSL:

dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

Aktifkan Virtual Machine Platform:

dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

3. Restart Komputer

Setelah mengaktifkan kedua fitur tersebut, restart komputer agar perubahan dapat diterapkan.

4. Set WSL2 sebagai Versi Default

Setelah sistem menyala kembali, buka PowerShell sebagai Administrator dan jalankan perintah berikut:

wsl --set-default-version 2

Perintah ini memastikan bahwa distribusi Linux yang diinstal selanjutnya akan menggunakan WSL2 sebagai versi default.

5. Instal Distribusi Linux

Instal distribusi Linux pilihan Anda melalui Microsoft Store:

  • Buka Microsoft Store.
  • Cari distribusi Linux (misalnya, Ubuntu).
  • Klik Dapatkan atau Instal untuk mengunduh distribusi tersebut.

6. Selesaikan Pengaturan Distribusi Linux

Setelah instalasi selesai:

  • Buka aplikasi Linux yang telah terinstal.
  • Ikuti petunjuk di layar untuk melakukan konfigurasi awal, seperti pembuatan akun pengguna dan pengaturan password.

7. Verifikasi Instalasi

Untuk memastikan WSL2 telah terpasang dengan benar, buka PowerShell dan jalankan:

wsl --list --verbose

Perintah ini akan menampilkan daftar distribusi Linux beserta versi WSL yang digunakan.

Troubleshooting

  • Kesalahan saat mengaktifkan fitur: Pastikan menjalankan PowerShell sebagai Administrator dan periksa kembali apakah sistem telah memenuhi prasyarat.
  • Distribusi tidak berjalan dengan WSL2: Pastikan perintah wsl --set-default-version 2 telah dijalankan. Jika perlu, ubah versi WSL untuk distribusi tertentu dengan perintah: 
    wsl --set-version <NamaDistribusi> 2
  • Referensi dan bantuan lebih lanjut: Jika menemui masalah, silakan kunjungi dokumentasi resmi instalasi WSL2 untuk informasi tambahan.
